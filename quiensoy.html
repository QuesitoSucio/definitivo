Mis proyectos

El Gusanito: Mi intento de recrear el clásico juego de la serpiente usando HTML, CSS y JavaScript. Aún estoy mejorando el movimiento y las colisiones.

Lista de Tareas: Un pequeño proyecto donde aprendí a manejar eventos en JavaScript. Permite agregar y eliminar tareas, pero aún quiero mejorar el diseño.

Galería de Imágenes: Una página donde practiqué el uso de CSS Grid y Flexbox. Las imágenes se ajustan automáticamente, pero quiero agregar efectos animados.

Mi Primera Calculadora: Un ejercicio para aprender a manipular el DOM. Puede sumar, restar, multiplicar y dividir, aunque el diseño aún es muy básico.
